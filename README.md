# react-boilerplate

React Boilerplate for effective development.

## Features
* Client & Server Hot Module Replacement (HMR)
* Server Side Rendering (SSR)

## Uasge
* Installation

`
yarn install
`

* Development

`
npm start
`
## Technologies
* React 15
  * react-helmet
* Express 4
* Webpack 3
  * hmr, react-hot-loader 3
  * webpack-merge
* Babel 6
  * babel-preset-env, babel-preset-stage-2, transform-decorators-legacy
